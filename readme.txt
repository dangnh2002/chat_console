chạy file setup.exe
sau khi cài đặt xong thì cửa sổ console hiện lên và bắt đầu phiên sử dụng

// hướng dẫn sử dụng
1. chạy file setup.exe
2. nhập tên hiển thị
3. chọn tạo kênh hoặc vào kênh chat bằng cú pháp
/yes tên_kênh mật_khẩu
/no tên_kênh mật_khẩu
4. thay đổi màu chữ hiển thị với cú pháp
/tên_màu
5. thoát kênh chat để vào kênh khác bằng cú pháp
/quit
6. tắt ứng dụng bằng cách tắt cửa sổ đang mở

//bảng mã màu có thể đổi được
màu trắng: /white
màu đỏ: /red
màu xanh: /blue
màu lục lam đậm: /darkcyan
xanh đậm: /darkblue
Màu xanh lá cây đậm: /darkgreen
